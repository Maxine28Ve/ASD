## Visits


| Visit      | Order                         |
|------------|-------------------------------|
| Pre-Order  | Root > L-Child > R-Child      |
| Symmetric  | L-Child > Root > R-Child      |
| Post-Order | L-Child > R-Child > Root      |
| BFS        | At each level, go from L to R |
